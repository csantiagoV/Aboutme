# NOVA SPARK

Portafolio profesional desarrollado con Vite, React, TypeScript y Tailwind CSS.

## Desarrollo local

1. Instala Node.js.
2. Ejecuta `npm install`.
3. Inicia el proyecto con `npm run dev`.

## Scripts disponibles

- `npm run dev`: servidor local de desarrollo.
- `npm run build`: compilacion para produccion.
- `npm run preview`: vista previa del build.
- `npm run lint`: revision de codigo.
- `npm test`: pruebas con Vitest.

## Publicacion en GitHub

1. Inicializa o conecta el repositorio Git.
2. Revisa que no subas `node_modules`.
3. Sube el codigo fuente una vez reemplaces tus datos personales finales.

## Publicacion en cPanel

1. Ejecuta `npm run build`.
2. Sube el contenido de `dist` a `public_html`.
3. Si usas rutas de React Router en el futuro, agrega una regla de reescritura en `.htaccess`.
